# Dotfiles

Requires `ghostty`, `hyprland`, `mako`, `nwg-look`, `rofi`, `rofi-calc`, `waybar`, `catppuccin-cursors-mocha` (AUR), and `catppuccin-gtk-theme-mocha` (AUR)

```sh
sudo pacman -S ghostty hyprland mako nwg-look rofi rofi-calc waybar
paru -S catppuccin-cursors-mocha, catppuccin-gtk-theme-mocha
```
